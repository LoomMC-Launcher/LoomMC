# 💎 LoomMC Launcher

The official custom Minecraft launcher for the **9LivesSMP** community. Built for performance, farming, and getting rich.

## 🚀 Features
* **One-Click Install**: Fast setup to get you into the game.
* **Pre-configured**: Optimized settings specifically for our server economy.
* **Farming Ready**: Built-in support for our farming and economy gameplay.

## 📥 How to Join
1. Go to the [Releases](https://github.com/Loom-Apps/LoomMC/releases) page.
2. Download the latest `LoomMC-Setup.exe`.
3. Run the installer and log in with your Minecraft account.
4. Click **Play** and you will be automatically connected to the server!
